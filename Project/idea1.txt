Idea mobile appName: "Whatever in BKK"อยากทำแอปที่ช่วยตัดสินใจเรื่องต่างๆในชีวิตประจำวัน เพราะเพื่อนจะทำไรทีชอบลีลา มัวแต่เม็ดConceptIdea: แอปช่วยตัดสินใจสำหรับชีวิตประจำวันผู้ใช้ใส่ตัวเลือกหลายอย่าง (เช่น “กินอะไรดี?”, “ไปไหนดี?”, “ทำงานไหนก่อน?”)แอปสุ่มเลือกคำตอบให้ → ลดเวลาลังเลหรือเถียงกับเพื่อนTarget Users: คนที่ลังเลง่าย, เพื่อนชอบรีรอ, คนชอบสนุกกับการสุ่มValue Proposition: ตัดสินใจเร็วขึ้น, สนุก, ใช้งานง่าย, Offline ได้ตัวอย่าง Scenario:“วันนี้กินอะไรดี?” → ใส่ตัวเลือก: ข้าวมันไก่ / ข้าวผัด / ก๋วยเตี๋ยวกด “Decide” → แอปเลือกให้MVP Featuresเวอร์ชันแรกควรโฟกัสที่ฟีเจอร์หลักที่ทำให้ใช้งานได้ทันทีAdd Options พิมพ์ตัวเลือกเอง หรือเลือกจากตัวอย่างที่แอปมีให้Decision / Random Choiceปุ่ม “Decide” → แอปสุ่มเลือกหนึ่งตัวเลือกเพิ่ม animation / effect ให้สนุก เช่น หมุนวงล้อHistory / Favoritesเก็บผลลัพธ์ที่เลือกแล้วสามารถดูย้อนหลังได้Simple UI / OfflineUI เรียบง่ายใช้งานได้โดยไม่ต้องเชื่อมเน็ตOptional Fun: Share result → แชร์ให้เพื่อนดูSet up PhaseStep 1: Research & Planศึกษาแอปคู่แข่ง เช่น “Decide Now!”, “Wheel of Names”กำหนดสี / Theme / UX แบบ playfulStep 2: Tech StackPlatform: Android (Android Studio + Kotlin + Jetpack Compose)Database: SQLite / Room (เก็บ History)Offline Ready: Core features ไม่ต้องเน็ตStep 3: DesignWireframe / Mockupหน้าจอหลัก:หน้าเพิ่มตัวเลือกหน้าเลือกผลแบบหมุนวงล้อหน้า HistoryStep 4: MVP Build & Testสร้างเวอร์ชันพื้นฐานทดสอบกับกลุ่มเพื่อนเล็ก ๆ → FeedbackNext Phase (หลัง MVP)Themes & Customizationเปลี่ยนสี / เอฟเฟกต์หมุนวงล้อ / avatar ของตัวเลือกSmart Suggestionsแอปเรียนรู้ตัวเลือกที่ใช้บ่อยแนะนำตัวเลือกตามเวลา / moodSocial / Multiplayer Modeเพื่อนร่วมใส่ตัวเลือกแล้วสุ่มพร้อมกันIntegrationเชื่อมกับ Calendar / Reminderสั่งให้แอปช่วย “Decide & Schedule”
